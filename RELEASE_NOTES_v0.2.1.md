# Release Notes - v0.2.1

Full-body Clawd animation update.

## Included

- full-body procedural Clawd drawings instead of face-only states
- pixel-body torso, arms, feet, and ground shadow
- state-specific props for typing, thinking, notification, sleep, yawn, happy, and error states
- same Claude Code serial and HTTP protocol as v0.2.0

## Note

This release does not embed the Clawd on Desk SVG assets directly. It redraws the character procedurally for the ESP32-S3 and ST7789 display.
